  7. GUI:-

    8. A graphical user interface (GUI) is a graphics-based interface that lets users interact with electronic devices using visual elements like icons, menus, and buttons.

  8. Web Browser :-

    9. A software application used to access information on the World Wide Web is called a Web Browser.

    When a user requests some information, the web browser fetches the data from a web server and then displays the webpage on the user's screen.

  9. web address :-

    10. It is also known as the URL Like the address for your home, a web address organizes information about a webpage's location in a predictable way.

  10. Domain :-

    1. Top Domain name :-

      2. TDL is the highest level of domain names on the internet, and it is everything that follows the final dot of a domain name. For example, in the domain name "google.com", ".com" is the TLD.

    2. domain name :-

      3. A domain name is a website's address that's used in a URL to identify which server a webpage belongs to.

    3. sub domain :-

      4. A subdomain is a prefix added to a domain name to separate and organize content on a website.

  11. URL :- Uniform Resource Locator

    12. URL (Uniform Resource Locator) is a unique address that identifies a resource on the internet, like a web page or file.

  12. URI:- Uniform Resource Identifier

    13. A URI (Uniform Resource Identifier) is a string that refers to a resource.

  13. Web page :-

    14. A web page is a document written in hypertext (also known as HTML) that you can see online, using a web browser.

  14. Website :-

    15. Most web pages include text, photos or videos, and links to other web pages. A group of many web pages managed by one person or company is a website.

  15. Hosting :-

    16. It’s a internet hosting service that hosts websites for clients, it offers the facilities required for them to create and maintain a site and makes it accessible on the World Wide Web. It simply means uploading information according to users request.

  16. server space :-

    17. Server space is the amount of storage space on a server that is leased out to clients to host their websites. It's also known as web space or disk space

  17. TCP/IP :-

    18. TCP(Transmission Control Protocol) meaning, is a communications standard for delivering data and messages through networks. TCP is a basic standard that defines the rules of the internet and is a common protocol used to deliver data in digital network communications.

  18. Protocol :-

    19. The most common meaning of protocol is “a system of rules that explain the correct conduct and procedures to be followed in formal situations,”

  19. HTTP :-

    20. HTTP (stands for Hypertext Transfer) Protocol, and is a set of rules that enables communication between a client and a server on the World Wide Web.

  20. HTTPs :-

    21. HTTPS stands for (Hypertext Transfer Protocol Secure), which is a secure version of the Hypertext Transfer Protocol (HTTP).

  21. SEO :- (Search Engine Optimization)

    22. using HTML tags to improve a website's visibility and ranking in search engine results.
    In simple terms, SEO means the process of improving your website to increase its visibility in Google, Microsoft Bing, and other search engines whenever people search for products you sell. It provides ranking based on different factors.
