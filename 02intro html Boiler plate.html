1. what is HTML.

    2. HTML HyperText Markup Language is a markup language that defines the structure and content of a web page. It's the most basic building block of the web and is used to tell web browsers how to display a page's words and images.

  2. what is Attribute.

    3. Attributes are used to adjust the behavior or display of an HTML element. It can be used to change the color, size or functionality of HTML element. For instance, in the above boiler plate lang, charset, name and content are attributes


    Elements in HTML have attributes these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.
