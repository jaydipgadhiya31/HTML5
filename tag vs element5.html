Element vs Tag.

Tag has no content written. For instance <h1 > </h1 >


Element has content written in between tags. <h1 > this is heading number 1</h1 >

Types of tags:-

There are 2 types of tags Opening and closing tags and self-closing tags. Here, meta tag is self closing tag.

(void , empty, null tag) { self closing tag } :-

area, base, br, hr , command, embed, col , img, input, and keygen.
